EssentialsX - El plugin esencial para tu servidor

EssentialsX es el plugin más completo y utilizado para servidores Minecraft. Proporciona una suite completa de comandos esenciales que todo servidor necesita, desde teletransportes hasta gestión de economía.

🎯 Características principales

✓Sistema de homes, warps y teletransportes
✓Economía integrada con Vault
✓Sistema de kits con recompensas
✓Mensajes de bienvenida y despedida
✓Sistema de cofres de mortero
✓Back después de muerte o teletransporte
💡

¿Sabías qué?

EssentialsX es el plugin más descargado de la historia de Minecraft, con más de 1.5 millones de descargas activas y soporte para versiones desde la 1.8 hasta la 1.20.4.

Comandos disponibles

/spawn

Teletransporta al punto de spawn del servidor

/home [nombre]

Teletransporta a tu casa

/sethome [nombre]

Establece un punto de casa

/warp [nombre]

Teletransporta a un warp público

/tpa [jugador]

Solicita teletransportarte a otro jugador

/tpaccept

Acepta una solicitud de teletransporte

/back

Regresa al último punto de teletransporte o muerte

/kit [nombre]

Recibe un kit

/msg [jugador] [mensaje]

Envía un mensaje privado

/balance

Muestra tu balance de dinero

/pay [jugador] [cantidad]

Transfiere dinero

/tpahere [jugador]

Solicita que otro jugador se teletransporte a ti

/tpdeny

Rechaza una solicitud de teletransporte

/hat

Pone el objeto que tienes en la cabeza

/afk

Marca como AFK (Ausente)

/seen [jugador]

Muestra la última vez que un jugador estuvo conectado

Archivo de configuración (config.yml)

# ===========================================
# ESSENTIALSX CONFIGURATION
# Versión: 2.20.1
# ===========================================

# IDIOMA Y LOCALIZACIÓN
locale: es

# TELETRANSPORTE
teleport:
  cooldown: 5
  delay: 3
  show-progress-bar: true
  cancel-on-move: true
  cancel-on-damage: true

# ECONOMÍA
economy:
  starting-balance: 100.0
  max-balance: 1000000.0
  currency-symbol: "$"
  currency-format: "{symbol}{amount}"
  currency-decimals: 2

# KITS
kits:
  starter:
    delay: 86400
    items:
      - "stone_sword"
      - "bread 16"
      - "wooden_pickaxe"
      - "iron_helmet"
      - "iron_boots"
  vip:
    delay: 43200
    items:
      - "diamond_sword"
      - "golden_apple 3"
      - "ender_pearl 8"

# HOMES
max-homes:
  default: 5
  vip: 10
  admin: 50

# MENSAJES
welcome:
  enabled: true
  message: "&aBienvenido &e{player} &aal servidor de InfiniHon!"
  sound: "ENTITY_PLAYER_LEVELUP"

quit:
  enabled: true
  message: "&c{player} &7ha abandonado el servidor"

# PROTECCIÓN
protect:
  prevent-spawn: false
  prevent-teleport: false

# MISCELÁNEO
misc:
  death-chest: true
  keep-inventory: false
  keep-exp: false
  afk:
    enabled: true
    kick-if-afk: false
    afk-timeout: 300
